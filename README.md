# Assistant client
The client for the assistant server.

# Setting up
Download the file `svr_speak.aia`

Import `svr_speak.aia` into MIT AI2 (http://ai2.appinventor.mit.edu)

Open the blocks menu, then change the global varaible "url" to your server endpoint.

# Building
Simple, open the `Build` menu on the top of AI2, then click `App ( save .apk to my computer )`. The .apk file will then download.
